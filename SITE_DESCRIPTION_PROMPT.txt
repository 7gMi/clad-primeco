PROMPT POUR REPRODUIRE LE SITE CLAD-PRIMECO EN HTML
=====================================================

Créez un site web professionnel pour "Clad-Primeco", une entreprise irlandaise spécialisée dans les systèmes de bardage (cladding). Le site doit être moderne, élégant et adaptatif (responsive).

STRUCTURE DU SITE
-----------------
Le site comporte 3 pages principales :
1. HOME (Accueil)
2. ABOUT (À propos)
3. CONTACT (Contact)

Il n'y a PAS de footer en bas des pages.

HEADER (EN-TÊTE) - Présent sur toutes les pages
------------------------------------------------

CARACTÉRISTIQUES :
- Position : Fixed en haut de page
- Comportement au scroll :
  * Avant le scroll (position initiale) :
    - Fond : dégradé noir transparent (from-black/60 via-black/30 to-transparent)
    - Logo plus grand
    - Texte "CLADDING PROFESSIONALS" visible sous le logo en blanc (police Georgia serif)
    - Navigation en texte blanc

  * Après le scroll (100px+) :
    - Fond : blanc avec ombre
    - Logo plus petit
    - Texte "CLADDING PROFESSIONALS" disparaît
    - Navigation en texte gris foncé

CONTENU DU HEADER :
- Logo : 3 versions responsive (mobile, tablette, desktop)
- Navigation (visible uniquement sur desktop/tablette) :
  * Home
  * About
  * Services (ancre #services)
  * Projects (ancre #projects)
  * Contact

COULEURS PRINCIPALES
--------------------
- Bleu principal : #2563eb (blue-600)
- Fond sombre : slate-900, gray-900
- Fond clair : blanc, slate-50
- Texte : slate-900 pour les titres, slate-700/600 pour le corps

=====================================================
DESIGN & ANIMATIONS
=====================================================

TYPOGRAPHIE :
- Titres : Police bold, tailles 4xl à 6xl
- Corps : Police régulière/medium, tailles lg à xl
- Taglines : Uppercase, tracking-wide, petites tailles

ESPACEMENT :
- Sections : padding 20 (py-20)
- Éléments : gaps de 6 à 16
- Container max : 7xl (max-w-7xl)

ANIMATIONS & TRANSITIONS :
- Hover sur boutons : changement de couleur + scale
- Hover sur cartes : ombre accentuée + zoom image
- Transitions : duration-300 ou duration-500
- Slider : fade in/out avec opacity

BORDURES & OMBRES :
- Cartes : shadow-lg, hover:shadow-2xl
- Inputs : border blanc transparent, focus:ring-2 bleu
- Coins arrondis : lg, xl, 2xl, 3xl selon les éléments

RESPONSIVE :
- Mobile first
- Breakpoints : md (768px), lg (1024px)
- Grids : 1 col mobile, 2-3 cols desktop
- Images : object-cover pour maintenir proportions
- Section contact : Layout vertical (flex-col) sur mobile, horizontal (flex-row) sur desktop
- Texte centré sur mobile, aligné à gauche sur desktop
- Gap augmenté entre éléments sur mobile (gap-8)

ICÔNES :
Utiliser lucide-react ou des icônes SVG similaires pour :
- ArrowRight, Building2, Layers, Shield
- Pencil, Hammer, CheckCircle
- Phone, Mail, MapPin
- Facebook, Instagram, Linkedin
- ChevronLeft, ChevronRight
