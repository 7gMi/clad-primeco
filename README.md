# Cladding & Roofing Professionals Website

A beautiful, production-ready slider website for cladding and roofing professionals.

## Required Images

Please add the following images to complete the website:

### Background Images (1280px × 720px)
Place in `public/images/backgrounds/`:
- `kingspan-panel.jpg` - Kingspan insulated panels background
- `architectural-cladding.jpg` - Architectural cladding background (Full HD)
- `aluminium-cladding.jpg` - Aluminium cladding background (Full HD)

### Logo Images
Place in `public/images/logo/`:
- `logo-desktop.png` (200px × 57px) - Desktop logo
- `logo-tab.png` (160px × 46px) - Tablet logo
- `logo-mobile.png` (140px × 40px) - Mobile logo
- `favicon.png` (32px × 32px) - Site favicon

## Features

- 3 automatic rotating slides with smooth transitions
- Fully responsive design (mobile, tablet, desktop)
- Manual slide navigation with circle indicators
- Professional navigation menu
- Call-to-action buttons
- Beautiful animations and hover effects
- Optimized for production
